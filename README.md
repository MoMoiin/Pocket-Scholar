# Pocket-Scholar
*A very bad portable e-ink flashcard and dictionary device.*

---

## Description
**Pocket-Scholar** is a very badly designed ESP32 + e-ink flashcard reader.  
The point of the project was simply to create *something* using C++ and hardware — and to learn along the way.

![Pocket-Scholar Prototype](https://github.com/user-attachments/assets/8c659e05-9166-4b66-bf42-8a4955e19657)

---

## Features

### Current
- Basic **e-ink + ESP32 integration**

### Planned
- Read **flashcards from SD card**
- Add an **algorithm** to show cards based on learning patterns
- Build a **basic menu system and settings**
- And… *whatever else I can come up with on the go*

---

## Dependencies
- [PlatformIO](https://platformio.org/)
- [ESP32](https://www.espressif.com/en/products/socs/esp32)
- e-ink display module

---

## Notes
This project is mostly for fun, experimentation, and learning embedded development.  
Don’t expect elegance — just curiosity, C++, and chaos.
